
# Multiple Statistical tests where run to analyse the data.
# First a T-test comparing Algorithm to Mean_Dev on the whole data set is run to validate there is a difference between sets
# Second cohen's d is calculated for Algorithm and Mean_Dev across the whole data set to calculate the overall effect size
# Thirdly a linear model regression test is run to see how the three factors Algorithm, Src_SubDiv and Input_Reduction effect Mean_Dev
# Fourth, a pratt analysis of the model is run to show the importance of each regressor.
#
# Finally, a t test is conducted for each expirmental test run. The only variable difference between the two data sets here is the algorithm being used.
# This is done to test if the Algorithm difference is universally better and also to highlight anomalies in the data set.
# Cohen's D is also calculated for each test run
